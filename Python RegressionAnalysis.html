Case Summary:

After three years serving customers across the San Francisco Bay Area, the executives at Apprentice Chef have come to realize that over 90% of their revenue comes from customers that have been ordering meal sets for 12 months or less. Given this information & dataset, they would like to better understand how much revenue to expect from each customer within their first year of orders. Thus, they have hired you on a full-time contract to analyze their data, develop your top insights, and build a machine learning model to predict revenue over the first year of each customers life cycle.

# setting outlier thresholds
dict_out_lo = {'AVG_CLICKS_PER_VISIT' :10, 'TOTAL_MEALS_ORDERED':20}
dict_out_hi = {'AVG_CLICKS_PER_VISIT' :18, 'TOTAL_MEALS_ORDERED':250}
response_var = 'REVENUE'

for key, value in dict_out_hi.items():
    colName = 'change_' + key
    colName_df = key
    df[colName] = 0
    condition_hi = df.loc[0:,colName][df[colName_df] > value]
    df[colName].replace(to_replace = condition_hi, value = 1, inplace = True)

for key, value in dict_out_lo.items():
    colName = 'change_' + key
    colName_df = key
    if colName not in df.columns:
        df[colName] = 0
    condition_lo = df.loc[0:,colName][df[colName_df] < value]
    df[colName].replace(to_replace = condition_lo, value = 1, inplace = True)
new_outliers = df.loc[:, df.columns.str.startswith('change_') |
